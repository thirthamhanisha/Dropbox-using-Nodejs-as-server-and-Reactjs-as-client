
Install ReactJS

On the terminal: npm install -g create-react-app

Follow the steps on the terminal to start the servers:

Back-end server(nodelogin)
	1. cd nodelogin
	2. npm install
	3. npm start

Front-end server(reactlogin)
	1. cd reactlogin
	2. npm install
	3. npm start
	
for testing with mocha:
front-end (nodelogin folder):
npm install mocha
npm install chai
npm install superset
npm install should --save-dev
